Message To Beginners

  • You don't need to be a genious to code

  • Math requirements vary depending on the type of coding

  • There isn't a perfect language or engine, use whatever gives you the best result and the one you prefer

  • Don't skip the theory, interiorize concepts

  • Practice a lot after you gained some knowledge

  • Don't rely solely on tutorials, use them to learn the basics and then start making projects on your own

  • If you are stuck try to find a solution in the docs or online, if you fail ask to qualified people inside a community

  • Learn to read and resolve an error message!
